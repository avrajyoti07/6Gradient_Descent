# 📉 Gradient Descent from Scratch

Implementation of **Gradient Descent** using only NumPy to train a Linear Regression model — no ML libraries used for training.

## How It Works
- Predicts `y = mx + b`
- Computes **MSE cost** and **gradients**
- Updates `m` and `b` each iteration until cost converges

## Hyperparameters
| Parameter | Value |
|-----------|-------|
| Learning Rate | `0.08` |
| Iterations | `10000` |

## Run It
```bash
pip install numpy matplotlib
jupyter notebook 3Gradient_descent.ipynb
```
